### 常量指针

const修饰指针

特点：指针的指向可以修改，但是指针指向的值不可以改

```cpp
int a = 10;
int b = 20;
const int * p = &a;

*p = 20; // 错误
p = &b;  // 正确
```

### 指针常量

const修饰常量

特点：指针的指向不可以改，指针指向的值可以改

```cpp
int a = 10;
int b = 20;
int * const p = &a;

*p = 20; // 正确
p = &b;  // 错误
```

### const既修饰常量又修饰指针

特点：指针的指向和指针指向的值都不可以改

```cpp
int a = 10;
int b = 20;
const int * const p = &a

*p = 20; // 错误
p = &b;  // 错误
```
