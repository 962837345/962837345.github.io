### TCP和UDP的比较

|              |                  TCP                   |                    UDP                     |
| :----------: | :------------------------------------: | :----------------------------------------: |
|   是否连接   |                面向连接                |                   无连接                   |
|   是否可靠   |    可靠传输，使用流量控制和拥塞控制    |    不可靠传输，不使用流量控制和拥塞控制    |
| 连接对象个数 |            只能是一对一通信            | 支持一对一，一对多，多对一和多对多交互通信 |
|   传输方式   |               面向字节流               |                  面向报文                  |
|   首部开销   |       首部最小20字节，最大60字节       |            首部开销小，仅8字节             |
|   适用场景   | 适用于要求可靠传输的应用，例如文件传输 | 适用于实时应用（IP电话，视频会议，直播等） |
